# Unity_Tetris
The Simple Tetris.

# Control
User can change it arbitrarily! <br>
## Basic setting
- Left : LeftArrow
- Left Rotate : Z
- Right : RightArrow
- Right Rotate : UpArrow
- Down : DownArrow
- Hard Drop : Space Bar
